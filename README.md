# DevLaunch 🚀

DevLaunch is a lightweight web sandboxing platform designed for developers who want to bypass the repetitive chore of setting up boilerplate code. Within seconds, users can customize, generate, and download ready-to-go starter templates for popular stacks as structured clean `.ZIP` archives.

The application communicates with a dedicated Node.js/Express API cloud server linked with Supabase to keep record of logs, project creation history, and secure user management.

---

## 🛠️ Tech Stack

### Frontend
* **React.js** (Single Page Application architecture)
* **React Router DOM** (Client-side route handling & private route protection)
* **Vanilla CSS / Inline Mapped Objects** (Modern fluid Dark Theme)

### Backend
* **Node.js** & **Express** (REST API architecture hosted on Render)
* **Supabase** (Database client wrapper for persistent storage)
* **Archiver** (Dynamic on-the-fly streaming engine for file compression)
* **Dotenv** (Environment variables management)
* **CORS** (Cross-Origin Resource Sharing handling)

---

## 🚀 Key Features

* **Instant Boilerplate Generation:** On-the-fly streaming compression directly via the server's RAM memory, preventing disk-write overhead and ensuring maximum download speeds.
* **Available Templates:**
  * **React Boilerplate:** Component architecture with a ready-to-use routing layer.
  * **Node.js REST API:** Pure Express implementation with CORS, JSON parser, and script commands pre-configured.
  * **Discord Bot Base:** Object-oriented automation frame leveraging discord.js configurations.
* **Persistent History Logs:** Full sync with Supabase PostgreSQL tables to track user project generations down to the second.
* **Secure Session States:** Complete responsive layout transition depending on valid browser `localStorage` credentials.

## 📦 API Architecture Reference

### Authentication Endpoints
* `POST /api/auth/cadastro` - Registers a new user inside the database.
* `POST /api/auth/login` - Validates secure user data and mounts browser tokens.
* `POST /api/auth/mudar-senha` - Allows remote updates to active account credentials.

### Project Workspace Endpoints
* `GET /api/projetos/lista/:usuario_id` - Fetches the collection array of historic structures generated by the specific developer.
* `POST /api/projetos/salvar` - Commits a log row to Supabase and immediately switches the HTTP pipe buffer to push a binary stream file attachment down to the client window browser.
