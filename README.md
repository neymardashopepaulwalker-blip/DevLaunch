# DevLaunch 🚀 | Stardance Hack Club Project

O **DevLaunch** é uma plataforma SaaS Fullstack projetada para acelerar o início de novos projetos. Ele permite que desenvolvedores configurem, visualizem e baixem templates prontos e estruturados (React Boilerplate, APIs em Node.js e Bots de Discord) com apenas um clique, contando com uma experiência visual premium e gerenciamento de conta em tempo real.

---

## 🎨 Funcionalidades Principais

- ⚡ **Gerador de Projetos Automatizado:** Download instantâneo de arquivos `.zip` customizados e compactados dinamicamente pelo backend.
- 🔍 **Visualizador de Código (Preview Mode):** Interface estilo VS Code integrada para inspecionar os arquivos do template antes do download.
- 🌓 **Suporte a Temas Dinâmicos:** Alternância fluida entre Modo Escuro (Dark Mode) e Modo Claro (Light Mode).
- 🔐 **Autenticação e Guarda de Rotas:** Fluxo seguro de Cadastro, Login e Proteção de Páginas para usuários não autenticados.
- 📜 **Histórico Sincronizado:** Painel integrado exibindo os projetos criados pelo usuário com persistência de dados.

---

## 🛠️ Stack Tecnológica

**Frontend:**
- React.js (Vite)
- Componentização nativa com Estilos Dinâmicos (CSS-in-JS)
- React Router DOM (Controle de Rotas e Segurança)

**Backend:**
- Node.js & Express
- Supabase (Banco de Dados PostgreSQL em nuvem para Auth e Projetos)
- Archiver (Geração física de arquivos binários `.zip` em tempo real)
