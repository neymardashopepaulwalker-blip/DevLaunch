# DevLaunch

DevLaunch is a tool that helps developers start projects faster.

I built it because I was tired of creating the same folder structure and configuration files every time I started a new project.

With DevLaunch, you can generate a project template and download it as a ZIP file in a few seconds.

## Features

- Generate project boilerplates
- Download projects as ZIP files
- Custom project names
- Simple and fast workflow

## Tech Stack

Frontend:
- React
- React Router

Backend:
- Node.js
- Express

Database:
- Supabase

## Challenges

The backend was the hardest part of this project.

Before building DevLaunch, I had very little experience with backend development. I spent a lot of time learning how APIs, databases, and file generation work together.

I also used AI tools when I got stuck, but I made sure to understand the solutions and learn from them.

## What I learned

- Backend development
- API creation
- Database integration
- ZIP file generation
- Full-stack application structure

## Run locally

Backend:

npm install
node server.js

Frontend:

npm install
npm run dev
